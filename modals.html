ML Real Estate Pipeline

Date: Apr 2025

Summary:

  • Connected to Zillow's API and gathered the raw Utah homes data
  • Explored the data and performed analysis of what columns to include and how to transform them if necessary
  • Added depth to the dataset with by engineering a feature that tracked how close each home was to significant geographic locations.
  • Tested various popular machine learning models and tuned hyperparameters to get the best results for each one. The best model was XGBoost.
  • Visualized results and wrote a paper on my findings.
